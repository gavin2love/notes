## 快速启动Grafna
```bash
mkdir -p /data/docker/grafana
sudo chown 472:472 /data/docker/grafana -R
docker run -d \
  -p 3000:3000 \
  --name=grafana \
  -v /data/docker/grafana:/var/lib/grafana \
  -v /etc/localtime:/etc/localtime:ro \
  --restart=always \
  --name grafana \
  grafana/grafana
```
> 浏览器访问 http://ip:3000 初始账号密码 admin / admin
