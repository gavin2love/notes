## 初始化
```bash
curl -s https://pealog.com/ftp/os7init.sh | bash -s `hostname`
```

## 禁用SELinux
```bash
sed -i '/SELINUX/s/enforcing/disabled/' /etc/selinux/config  && setenforce 0 
```

## CentOS7换源-国内
```bash
# 大陆
curl -o /etc/yum.repos.d/CentOS-Base.repo https://mirrors.aliyun.com/repo/Centos-7.repo
curl -o /etc/yum.repos.d/epel.repo https://mirrors.aliyun.com/repo/epel-7.repo
yum clean all && yum makecache

# 香港
curl -o /etc/yum.repos.d/CentOS-Base.repo https://mirrors.aliyun.com/repo/Centos-7.repo
curl -o /etc/yum.repos.d/epel.repo https://mirrors.aliyun.com/repo/epel-7.repo
sed -i 's/mirrors.aliyun.com/mirror.xtom.com.hk/' /etc/yum.repos.d/CentOS-Base.repo
sed -i 's/mirrors.aliyuncs.com/mirror.xtom.com.hk/' /etc/yum.repos.d/CentOS-Base.repo
sed -i 's/mirrors.cloud.aliyuncs.com/mirror.xtom.com.hk/' /etc/yum.repos.d/CentOS-Base.repo
sed -i 's/mirrors.aliyun.com/mirror.xtom.com.hk/' /etc/yum.repos.d/epel.repo
yum clean all && yum makecache
```
